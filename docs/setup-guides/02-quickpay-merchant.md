# 2. Create a Quickpay merchant and copy keys

1. In Quickpay Manager, create a **Merchant** account for your shop.
2. Open **Settings → Integration**.
3. Copy into the setup form on `payment.rns-apps.dk/{slug}/admin/setup`:
   - **Merchant ID**
   - **Private key**
   - **Payment Window** user API key (use the restricted Payment Window user, not the owner key)

4. Click **Save** and confirm **Quickpay: Connected**.
