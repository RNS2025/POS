# 3. Clearhaus (you negotiate directly)

1. In Quickpay Manager: **Settings → Acquirers → Clearhaus** — apply for an agreement.
2. **Negotiate terms directly with Clearhaus.** RNS-Apps is not part of this contract.
3. When approved, Clearhaus appears active in Quickpay Manager.

You can test payments in Quickpay test mode before Clearhaus is approved.
